# Running the project


**Get source code**
```bash
git clone https://github.com/crownest/helping-hand-web.git (Use HTTPS)
git clone git@github.com:crownest/helping-hand-web.git     (Use SSH)
```

**Change directory and branch**
```bash
cd helping-hand-web
git checkout staging
```

**Install packages**
```bash
npm install
```

**Run project**
```bash
npm start
```
